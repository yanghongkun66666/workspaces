Go 秒杀演进实验室

这套页面现在走真实 MySQL 数据库，方便你直接对照 Java 版看两个语言里 `select / update / insert` 的并发行为有什么共性。

连接方式

如果你是从 IDEA 或其他静态预览页打开本页面，`/api/...` 的相对路径会打到预览服务器本身，从而得到 404。把后端地址填成 `http://localhost:8090` 后再点保存即可。浏览器里的 `304` 只是静态资源缓存命中，不是后端错误。

超卖实验

这两个按钮都走真实数据库。朴素版先 `select stock` 再回写，乐观锁版执行 `update ... where stock > 0`。

一人一单实验

使用相同 userId 并发请求，观察不加锁与用户粒度锁的差别。

异步落库实验

接口先受理请求并放进队列，再由后台 goroutine 把订单写进数据库。这里的队列用来模拟 Redis Stream / MQ。

实验输出

重点看 `oversold`、`duplicateOrder`、`storage`、`dbOrders`、`acceptedOrders`、`persistedOrders`。
